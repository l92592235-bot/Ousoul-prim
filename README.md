# أصول برايم (Osoul Prime)

منصة خاصة متعددة العملاء لإدارة الأملاك العقارية والعقود، مبنية بـ Next.js + Postgres (Neon).

## متغيرات البيئة المطلوبة على Vercel
- `DATABASE_URL` — رابط الاتصال بقاعدة بيانات Neon (مع `sslmode=require`).
- `ADMIN_SECRET` — كلمة مرور طويلة عشوائية لحماية لوحة `/admin` التي تُستخدم لإنشاء روابط عملاء جدد.

## البنية
- `/` — صفحة عامة مقتضبة (لا تكشف أي معلومة).
- `/admin` — لوحة خاصة بمالكة المنصة لإنشاء رابط فريد لكل عميل جديد (`/c/{token}`) ومتابعة الحسابات.
- `/c/{token}` — تطبيق العميل: عند أول زيارة يُطلب إعداد كلمة السر والبريد وواتساب، وبعدها تسجيل الدخول بكلمة السر فقط.

## الأمان
- كلمات السر مخزّنة عبر scrypt (ملح عشوائي لكل حساب).
- الجلسات: رمز عشوائي طويل (32 بايت) مخزّن في كوكي httpOnly + secure، صلاحيته 24 ساعة، ويُتحقق منه من جدول `sessions` في كل طلب.
- قفل الحساب تلقائيًا بعد 5 محاولات دخول خاطئة لمدة 15 دقيقة.
- عزل كامل للبيانات بين العملاء عبر `client_id` في كل جدول.
